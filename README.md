### mlai-berkeley-pracapp-coupons
The objective of the project is to analyze "coupon.csv" file
This file has the following details:
1. Customer
2. Coupon
3. Restaurants
4. Temperature
5. Likeliness to go a particular type of Restaurant

We have to analyze the details and figure out what is the 
Possibility that someone will accept a coupon offered

We will be analyzinf the data and will understand:
1. How many people have accepted coupons
2. Which Gender accepted more coupon
3. Which type of Restaurant are people more interested in
4. Does coupon for Bar intrigue all type of drivers
5. Does temperature/weather affect acceptance
6. Are people without partners interested if Restaurant coupons are offered
7. Are people willing to take a long detour in the opposite direction


Final Objective:
There are endless dimension to analyze a data
But this exericse gives an opportunity understand how we can use the data
and getting details that will be used to target more successfully

In future the final goal will be to build a model to identify the best
Combination of customer, type of Restaurant, time, weather, income and
other factors to predict the best chances for customer accepting a coupon
without human intervention
